# AI Fitness Tracker

A Next.js fitness tracking app that uses Groq AI to calculate calories burned based on workout type and duration.

## Features

- Calculate calories burned using AI (Groq API)
- Track workout history with dates
- Local storage for workout data
- Clean, modern UI with Tailwind CSS

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env.local` file with your Groq API key:
   ```
   GROQ_API_KEY=your_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Groq API (Llama 3.3 70B)
